# Frontend Project

본 프로젝트는 프론트엔드 과제 제출용 프로젝트입니다.  
README에 기재된 방법대로 진행하면 **별도의 추가 설명 없이 바로 실행 가능**합니다.

## 실행 방법

### 1. 프로젝트 클론
```bash
git clone https://github.com/username/project-name.git
cd project-name
```
### 2. 의존성 설치
```bash
npm install
```
### 3. 프로젝트 실행
```bash
npm run dev
```
## 접속 방법
